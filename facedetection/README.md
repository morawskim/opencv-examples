Facedetection is simple C++ program, which detects faces. Required openCV library.

Compiling
===========================
Download git repo

    git clone  

Go to build dir

    cp ./build/
    
Run cmake

    cmake ..

Compile

    make
    
Add run

    opencv_facedetection path/to/image/with/people.jpg
